# PowerBI-Healthcare-Analytical-Project-

An interactive and insight-driven Power BI dashboard built to analyze hospital performance data using advanced tools like Power Query, Data Modeling, and DAX Measures. This project focuses on transforming raw healthcare data into meaningful insights for performance monitoring and decision-making.

(A)  Project Overview

This dashboard provides detailed analysis on:

• Total Patients Treated

• Total Revenue Generated

• Average Patient Satisfaction Score

• Department-wise Patient Distribution (Cardiology, Neurology, Orthopedics, Pediatrics)

• Doctor-wise Workload Analysis

• Departmental Satisfaction Comparison


(B) Tools & Features Used

 • Power Query – Data cleaning and transformation

 • Data Modeling – Relationship building between tables

 • DAX Measures – KPIs like Total Revenue & Revenue per Patient

 • Interactive Visuals – Bar charts, KPI cards, Gauge chart

 • Filters/Slicers – Department-level dynamic filtering

 • Clean Dashboard Design – Professional healthcare-themed layout


(C) Key KPIs Tracked

• Total Revenue: 28M

• Total Patients Treated: 5,068

• Average Satisfaction Score: 3.53

• Doctor Performance & Workload Distribution


(D) Use Cases

Healthcare performance monitoring

Department-level comparison

Doctor productivity analysis

Patient satisfaction tracking

Data-driven hospital management decisions
